Ubihelper - documentation index

Chris Greenhalgh, 2012-03-09.

This application provides an Android service which runs in the background on the phone and allows other applications - in particular applications created using AppFurnace - to access sensors on this device, and potentially to link to other devices.

End-user documentation

For people who are using an application developed using ubihelper.
  • UbihelperEndUserGuide.html

Application-developer documentation

For people who are developing (e.g.) AppFurnace applications using ubihelper.

  • UbihelperApplicationDeveloperGuide.html

Technical documentation

For people who are extending ubihelper itself or wish to inter-operate with it.

  • UbiHelperPeerNetwork.html - documentation (such as it is) of the network protocol used to talk to other devices and servers.
